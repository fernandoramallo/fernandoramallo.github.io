Doodle Animation File

About

When you save an animation in Doodle Studio 95! it’ll be saved in a custom format called a Doodle Animation File.

When the Project window isn’t zoomed in, the Doodle Animation files show up with an onion icon:

File

Click an Animation File to see a preview and some options:

  • Edit
    Open the animation in Doodle Studio 95!

  • Create…
    Import the animation into your scene, creates all the necessary objects and components.
    • Make Sprite:
      Creates an object with a default Sprite renderer.
    • Make 3D Sprite:
      Makes a Sprite renderer that casts shadows, to make 2.5D worlds.
    • Make UI Image:
      Makes an Image inside of a Canvas using the Unity UI system.
    • Make Particles:
      Makes a particle system.

  • Convert to…
    Convert the animation into a different file format.
    • Save As Sprite Sheet:
      Saves the animation as a texture sheet, and imports each frame as a Sprite.
    • Save As GIF:
      Saves the animation as a GIF file.

Settings

Sets the default settings for the animation. These could be overriden in the Doodle Animator component.

  • Sprite Border
    Edits the left, bottom, right and top edges of the resulting Sprite border, in pixels. Use this to add 9-slicing to UI elements.
