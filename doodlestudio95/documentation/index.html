DOODLE STUDIO 95 is a drawing and animation tool for Unity.

MS Paint meets Macromedia Flash meets Unity, focused on making work

Get Doodle Studio 95! at itch.io!

What’s it for

  • Doodle an animation and add it to your game with one click
  • Turn your drawings into sprites, UI elements, particles or textures seamlessly
  • You don’t need to be an artist!

Features

  • Five Symmetry modes
  • Easy presets for backgrounds, characters, 9-sliced UI, playing cards
  • Draw and animate inside the Scene View
  • Customize and save color palettes
  • Easy API for playback, control animations with scripts
  • Don’t think about materials or image formats, it Just Works.
  • Record mode adds frames as you draw
  • Experimental features!
  • Record a sound for your animation. Boop!
  • Convert to sprite sheets or GIFs
