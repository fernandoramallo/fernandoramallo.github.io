Preferences

About

You can customize Doodle Studio 95! by editing the Preferences file in the Doodle Studio 95 folder:

Settings

  • Save Folder
    Where should the animations be saved by default, relative to your projectâ€™s Assets folder.
  • Presets
    Settings for each preset in the New window.
  • Brush Height
    Brush sizes for each of the size levels.
    The values are from 0 to 1 where 1 is the total height of the animation file.
  • Palettes
  • Beta
    Features that are mostly working but might have some quirks.
    • Scene View Drawing
      Lets you draw directly on the scene view. See Drawing in the Scene View (beta)
    • Show Jumble tool
      Adds the Jumble tool to the tool bar in the Drawing Window. Use Jumble to randomly move pixels around your brush and create gradient effects.
  • Experimental
    Features that are currently in development. Use at your own risk!
    May require closing and reopening the Drawing Window.
    • Sound Recorder
      Lets you record a sound with your microphone and associate it with your animation automatically.
    • Custom Brush
      Choose a texture to use as a brush.
